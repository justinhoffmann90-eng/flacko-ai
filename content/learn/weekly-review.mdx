## Why Weekly Reviews?

Trading isn't just about individual days. The **weekly review** helps you:

| Benefit | How |
|---------|-----|
| **See the bigger picture** | Daily noise fades, patterns emerge |
| **Track accuracy** | Did levels hold? Did scenarios play out? |
| **Improve over time** | Learn from what worked and didn't |
| **Prepare for next week** | Fresh levels, new setup |

---

| Followed mode cap? | Yes / No |

### Part 3: Lessons

| What Worked | What Didn't |
|-------------|-------------|
| | |
| | |

## Weekly Review Checklist

### ✅ Before You Start
- [ ] Close all positions (or document open)
- [ ] Pull up weekly chart
- [ ] Have daily reports from the week

### ✅ Analysis Steps
- [ ] Mark the week's high and low
- [ ] Note which levels were tested
- [ ] Review each day's mode vs outcome
- [ ] Calculate your win rate
- [ ] Identify your best and worst trade

### ✅ Preparation
- [ ] Note key levels for next week
- [ ] Check OpEx calendar
- [ ] Review upcoming catalysts
- [ ] Set mental goals for next week

## Level Accuracy Tracking

| Level | Forecast | Actual | Held? |
|-------|----------|--------|-------|
| Call Wall | $430 | Tested at $428 | ✓ |
| Gamma Strike | $420 | Bounced at $418 | ✓ |
| Put Wall | $400 | Not tested | — |
| Master Eject | $390 | Not tested | — |

**Accuracy rate:** 2/2 tested levels held = 100%

---

## Scenario Accuracy

| Week | Forecast | Outcome | Correct? |
|------|----------|---------|----------|
| This week | Base 60% | Ranged $415-$428 | ✓ |
| Last week | Bull 35% | Broke out to $430 | ✓ |
| 2 weeks ago | Bear 25% | Dropped to $400 | ✗ (called Base) |

**Track this over time to calibrate your reads.**

## Common Mistakes

| Mistake | Fix |
|---------|-----|
| Skipping the review | Schedule it — Friday close or weekend |
| Only reviewing winners | Losers teach more |
| Not tracking accuracy | Data beats feelings |
| No action items | Every review should have takeaways |

---

## Key Takeaways

| Principle | Remember |
|-----------|----------|
| **Review weekly** | Not daily, not monthly |
| **Track levels** | Did they hold? |
| **Calculate accuracy** | Over time, trends emerge |
| **Set goals** | What will you do better? |
